# Gemini Invoice Analyzer 🧾

A Streamlit web application that uses the Google Gemini API to analyze invoices. Simply upload an image of an invoice, enter a prompt (e.g., "What is the total amount?", "List the items purchased"), and the app will extract the information for you.

## Features
- Upload invoice images (JPG, JPEG, PNG).
- Custom prompt queries to extract specific data using Gemini 2.5 Flash.
- Simple, interactive user interface powered by Streamlit.
